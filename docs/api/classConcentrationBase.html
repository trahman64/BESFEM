Detailed Description

Abstract base class for BESFEM concentration solvers.

ConcentrationBase defines the shared interface and storage used by all concentration-update modules, including solid-particle concentration and electrolyte concentration solvers.

Derived classes must implement:

  • SetupField()
  • UpdateConcentration()

This class stores common mesh data, finite element spaces, element volumes, diagnostic quantities, and temporary MFEM workspaces used during concentration updates.
