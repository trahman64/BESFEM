Detailed Description

Abstract base class for BESFEM electric-potential solvers.

PotentialBase defines the shared interface and common FEM data used by electrode and electrolyte potential solvers. Derived classes initialize potential fields, assemble material-dependent systems, solve the finite element equations, and update boundary voltages when needed.
