Detailed Description

Solves electrolyte salt transport in BESFEM battery simulations.

ElectrolyteDiffusion updates the electrolyte concentration field using a diffusion equation with electrochemical reaction coupling at electrode/ electrolyte interfaces. It also provides a salt-conservation correction used to maintain the electrolyte inventory during time integration.
