Detailed Description

Owns the runtime solvers and grid functions for a BESFEM simulation.

SimulationState is the main container for all dynamically allocated solver objects and MFEM fields used during a run. It stores single-electrode fields, electrolyte fields, particle-resolved fields, reaction fields, pairwise chemical-potential workspaces, and output workspaces.
