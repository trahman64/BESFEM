Detailed Description

Stores all user-configurable simulation settings.

SimulationConfig contains the parameters required to initialize and run a BESFEM simulation, including mesh information, battery configuration, material assignments, initial conditions, and numerical solver settings.

The structure is populated from the configuration file and command-line arguments before the simulation begins.
