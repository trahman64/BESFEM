Detailed Description

Stores geometry-dependent fields and global quantities used throughout BESFEM.

Domain_Parameters constructs and manages the phase-field masks, interface fields, element volumes, and global integrals required by the concentration, potential, and reaction solvers.

◆ SetupDomainParameters()

void Domain_Parameters::SetupDomainParameters ( const char *  mesh_type)

Initialize all domain parameters based on the mesh type.

Steps:

  • Allocates and zeros ψ, ψₑ, ψ_A, ψ_C, AvP, AvA, AvC, AvB
  • Projects/interpolates distance-function-based fields
  • Computes element volumes (EVol)
  • Integrates ψ and ψₑ to compute gtPsi, gtPse
  • Computes global target current gTrgI
Parameters
mesh_typeCharacter flag identifying the mesh geometry:
  • "ml" MATLAB mesh
  • "v" voxel-derived mesh
